# E-commerce Microservices Platform

Proyecto de comercio electrónico basado en arquitectura de microservicios.

## Tecnologías

- Docker
- REST API
- Microservices
- Base de datos relacional

## Servicios

- User Service
- Product Service
- Order Service
- Inventory Service
- Payment Service
